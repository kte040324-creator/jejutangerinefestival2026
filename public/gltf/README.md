# GLTF Models for Hero Section

Put your **.glb** (or .gltf) files here with these exact names:

- **g1.glb**
- **g2.glb**
- **g3.glb**
- **g4.glb**

The hero section loads these four files and displays them with the same positions and floating animation as the original design. No placeholder content — only your g1–g4 models are used.
